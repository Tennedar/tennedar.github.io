// ++自增1 --自減1

        int d=25;

        System.out.println("b%a="+(b%a));

//與 & 或 | 異或 ^ 非 ~
//按位左移 << 右移>> 右移+補零 >>>
public class Test{
    public static void main(String[] args){
        int a=60;
        int b=13;
        int c=0;
        c=a&b;
        System.out.println("a&b="+c);
        c=a|b;
        System.out.println("a|b="+c);
        c=a^b;
        System.out.println("a^b="+c);
        c=~a;
        System.out.println("~a="+c);
        c=a<<2;        System.out.println("a<<2="+c);        c=a>>2;
        System.out.println("a>>2="+c);
        c=a>>>2;
        System.out.println("a>>>2="+c);
    }
}

// 與&& 或|| 非！
public class Test{
    public static void main(String[] args){
        int a=5;
        boolean b=(a<4)&&(a++<10);
        System.out.println("a="+a);
    }
}
/*
解析： 该程序使用到了短路逻辑运算符(&&)，首先判断 a<4 的结果
为 false，则 b 的结果必定是 false，所以不再执行第二个操作
a++<10 的判断，所以 a 的值为 5。
*/

#運算符2
/*条件运算符（?:）
条件运算符也被称为三元运算符。该运算符有3个操作数，并且需要判断布尔表达式的值。该运算符的主要是决定哪个值应该赋值给变量。 */
public class Test{
    public static void main(String[] args){
        int a,b;
        a=10;
        b= (a==1)? 20:30;
        //若 a=1 則b=20，反之為30
        System.out.println("b="+b);
        b=(a==1)?40:50;
        System.out.println("b="+b);
    }
}

java学习笔记3

java的兩大數據類型

java的兩大數據類型

內置數據類型

  • byte

    8位、有符号 -128~127 默认0
    byte类型用在大型数组中节省空间，因为byte占用的空间只有int的1/4

  • short

    16位，有符号 默认0 -2^15~2^15-1
    一个short是int所占空间的1/2

  • long

    64位、有符号 默认0L -2^63~2^63-1
    >>理论上不分大小写，但是若写成>>容易与数字1混淆，不容易分辩。所以最好大写。

  • int

    32位、有符号 默认0 -2^31~2^31-1

  • float

    单精度、32位，默认0.0f，不能用来表示精确的值eg货币

  • double

    双精度、64位，不能用来表示精确的值eg货币。默认0.0d
    浮点数的默认类型为 double 类型；

7 是一个 int 字面量，而 7D，7. 和 8.0 是 double 字面量。

Float和Double的最小值和最大值都是以科学记数法的形式输出的，结尾的"E+数字"表示E之前的数字

  • boolean

    默认False
    好像是小写？？？？？

  • char

    16位
    最小值\u0000（十进制=0）（默认值）；最大值\uffff（十进制=65535）可以储存任何字符

  • String(or any object)

    默认值null

对于数值类型的基本类型的取值范围，我们无需强制去记忆，因为它们的值都已经以常量的形式定义在对应的包装类中了。

引用數據類型

引用类型指向一个对象，指向对象的变量是引用变量。这些变量在声明时被指定为一个特定的类型，比如 Employee、Puppy 等。变量一旦声明后，类型就不能被改变了。

对象、数组都是引用数据类型。

所有引用类型的默认值都是null。

一个引用变量可以用来引用任何与之兼容的类型。

java常量

常量在程序运行时是不能被修改的。

在 Java 中使用 final 关键字来修饰常量，声明方式和变量类似：

final double PI = 3.1415927;

为了便于识别，通常使用大写字母表示常量

当使用字面量的时候，前缀 0 表示 8 进制，而前缀 0x 代表 16 进制
